Meus Projetos

  • Clone da Landing Page do Disney+ O Clone Disney+ é um projeto que replica a interface da plataforma Disney+ com foco em desenvolvimento front-end. Ele utiliza HTML, CSS (Sass/SCSS), JavaScript e a ferramenta Gulp para otimização. A estrutura do HTML é semântica, o CSS é bem organizado com a metodologia BEM, e o JavaScript adiciona interatividade, como guias e acordeões de FAQ. A automação de tarefas é gerenciada pelo Gulp para compilar Sass e minificar scripts, facilitando o desenvolvimento e otimização. A organização do código é mantida por meio de classes CSS, tornando-o modular e reutilizável. As seções interativas são aprimoradas com eventos do DOM, permitindo mostrar ou ocultar conteúdo com facilidade.
  • Landing page Essa Landing Page é de um site fictício de viagens e turismo. Utilizei HTML, CSS, JavaScript, Bootstrap e jQuery para criar uma experiência interativa e envolvente para os visitantes. O projeto inclui funcionalidades de navegação, carrossel de imagens, informações detalhadas sobre destinos e um formulário de contato.
  • Website Este projeto é uma página web que usa Bootstrap para criar um layout responsivo com um carousel de imagens e cards de conteúdo. As tecnologias usadas são HTML, CSS e JavaScript. As funcionalidades incluem um carousel de imagens, cards de conteúdo responsivos e sobreposição de texto nas imagens.
  • Galeria de fotos Este é um projeto simples de galeria de fotos desenvolvido em HTML, CSS e JavaScript. A galeria permite ao usuário visualizar imagens existentes e adicionar novas imagens com URLs fornecidos.
  • Loja virtual A Clothes Shop é um site fictício de comércio eletrônico responsivo, criado com HTML, CSS e JavaScript. Seu design é flexível e adaptável, garantindo uma experiência agradável para os usuários em vários dispositivos.

Tecnologias que uso, passe o mouse para saber mais!

  • HTML5Linguagem de marcação padrão para estruturar o conteúdo de páginas da web, introduzindo elementos semânticos para melhor acessibilidade e suporte a multimídia.
  • CSS3Versão mais recente do CSS que oferece recursos avançados de estilo, como transições, animações, sombras, gradientes e flexbox.
  • JavaScriptLinguagem de programação amplamente usada para adicionar interatividade e dinamismo às páginas da web, além de criar aplicativos da web complexos.
  • GruntAutomatiza tarefas de desenvolvimento, como minificação de arquivos, compilação de código, otimização de imagens e muito mais.
  • SassPré-processador CSS que permite aninhar seletores, criar variáveis, mixins e funções, tornando o código CSS mais modular e legível.
  • GulpAutomatizador de tarefas JavaScript que simplifica o processo de desenvolvimento, incluindo tarefas como minificação, compilação de código, otimização de imagens e muito mais.
  • BootstrapFramework CSS que oferece uma coleção de componentes e estilos prontos para criar interfaces web responsivas e atraentes.
  • ReactBiblioteca JavaScript para construção de interfaces de usuário de página única, permitindo a criação de aplicativos web reativos e eficientes.
